Help

Short, friendly instructions. No tech terms needed.

What this app does

  • Add items with a title, an amount, and an optional link.
  • Everything saves on your device and works offline.
  • You can Export to a file and save it to Google Drive/Dropbox.
  • You can Import that file on any device to restore your items.

How to install

  • Android / Chromebook / PC (Chrome/Edge): Open the app link → click the install icon in the address bar (or menu → Install).
  • iPhone/iPad: Open in Safari → Share → Add to Home Screen.

How to back up

  1. Tap Export. A file named nzbp-backup.json will download.
  2. Save that file in a safe place (Google Drive, Dropbox, or email it to yourself).
  3. To restore on any device: tap Import and pick that file.

Tips

  • Use the color circles at the top to switch themes. Your choice is remembered.
  • You can click “Open” in the table to visit item links.
  • The app works without internet once it’s been opened at least once.
